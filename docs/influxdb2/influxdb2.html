influxdb2.connect(url, token)[source]¶

Get an InfluxDB “connection” object.

Parameters

  • url (str) – Base URL to your InfluxDB instance. Example: https://localhost:8086/

  • token (str) – Valid token for your user to log in with. Example: abcdef1234567890

Returns

influxdb2.core.Influx instance.
