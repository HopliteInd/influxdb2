exception influxdb2.exceptions.InfluxAPIError(status, code, message, op, err)¶

Bases: influxdb2.exceptions.InfluxError

Exception raised for errors from the InfluxDB API.

Parameters

  • status (int) – HTTP Status code generating the error.

  • code (str) – Status code from InvluxDB’s API.

  • message (str) – Human readable message describing the error.

  • op (str) – Describes the logical code operation during error. Useful for debugging.

  • err (str) – A stack of errors that occurred during processing of the request. Useful for debugging.
