influxdb2.types module¶

Defining custom types and type enforcement helpers.

influxdb2.types.ID_REGEX = re.compile('^[a-f0-9]{16}$')¶

Regex to validate a string is a syntactically valid ID in InfluxDB.

influxdb2.types.ensure_id(src, name, value)¶

Ensure a variable is a syntactically valid id within InfluxDB

Parameters

  • src (str) – Source function name.

  • name (str) – Variable name for the id in value.

  • value (str) – Object to validate that it’s of the correct ID syntax.

Raises

ValueError – If the value is not a valid id.

influxdb2.types.ensure_type(src, name, value, obj_type)¶

Ensure the type cast of an object.

Parameters

  • src (str) – Source function name.

  • name (str) – Variable name for the object in value.

  • value (Any) – Object to ensure the type of.

  • obj_type (Type[Union[Any, Tuple[type, type]]]) – Type of object that value must be. This may be a tuple of types as well.

Raises

ValueError – If the type does not match obj_type
